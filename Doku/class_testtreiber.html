void Testtreiber::execute_tests ( )
static

Erstellt ein Objekt der Klasse Testtreiber, anhand von dem anschließend die Klasse Heap mit vier verschiedenen Datentypen getestet wird (int, float, double, char). Die künstlich erzeugten Scopes {} sind dazu da, um keinen Speicherplatz zu verschwenden. Der Heap eines bestimmten Datentyps wird zerstört, sobald der Scope verlassen wird.
